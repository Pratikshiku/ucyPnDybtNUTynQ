## Vaccine Release and Vaccination Appointment System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/0f03dcd4036e428999a78d39218f210b/ghb20250916) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript` etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): Regular User, Doctor, Administrator.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Forum
5. Announcement Information
6. Vaccine Information
7. Hospital Information
8. Personal Center
9. Change Password
10. Forum Management
11. Announcement Information
12. Vaccine Information
13. Hospital Information
14. Vaccination Appointment Management


**Doctor Functions:**
1. Login
2. Register
3. Change Password
4. Personal Information
5. Forum Management
6. Announcement Information Management
7. Vaccine Information Management
8. Hospital Information Management
9. Vaccination Appointment Management



**Administrator Functions:**
1. Login
2. Personal Center
3. Administrator Management
4. Announcement Type Management
5. Vaccine Type Management
6. Hospital Type Management
7. Forum Management
8. Announcement Information Management
9. Vaccine Information Management
10. Doctor Management
11. Hospital Information Management
12. User Management
13. Vaccination Appointment Management
14. Carousel Information
